Rekord Help Center

On-chain proof of attendance, made simple.

Rekord allows organizations to issue verifiable, blockchain-based badges as NFTs. These badges act as permanent proof that a participant attended or contributed to an event.

Unlike traditional certificates, Rekord badges:

  • Are minted directly to the participant’s wallet
  • Cannot be duplicated or forged
  • Exist permanently on the blockchain
  • Require zero crypto knowledge from users

Think of Rekord as certificates upgraded for Web3.

How Rekord Works

  1. Organization creates an event
  2. Rekord generates secure claim links
  3. Participants receive links via email
  4. Participants connect wallet and mint badge
  5. Badge is stored permanently on-chain

Rekord automatically handles NFT generation, metadata creation, IPFS uploads, gas fees, and mint security.

Creating an Event

Creating an event defines how your badge behaves, how it looks, and who can claim it.

Basic Event Information
  • Event Name – Displayed on the NFT
  • Description – Explains what the badge represents
  • City & Date – Stored in metadata
  • Event Image – Base image used for NFT generation

Tip: Use a high-quality square image (1024*1024 for non cropped images) for best results.

Event Types

Physical Events

Use physical events for real-world gatherings such as meetups, hackathons, and conferences.

  • Organizer defines participant count
  • One claim link per participant
  • Each link is single-use
Virtual Events

Virtual events are ideal for online sessions such as webinars and live streams.

  • Links may be shared more freely
  • Each wallet can mint only once

Event Prestige Levels

Prestige defines the visual identity and importance of your badge.

Standard

Clean, minimal design for regular events.

Signature

Premium styling for important or sponsored events.

Flagship

Highest prestige tier for major conferences or annual events.

Loyality System

Rekord dynamically upgrades NFTs based on participant history.

  • First-time participants receive muted or grayscale badges
  • Returning participants unlock visual enhancements
  • Loyal participants receive full-color badges with wallet imprint

Loyalty is wallet-based, not email-based.

Claiming a Rekord Badge

  1. Open your claim link
  2. Connect wallet
  3. Preview your NFT
  4. Enter secret code (if required)
  5. Mint badge

No gas fees are required from participants.

Troubleshooting

  • NFT not visible immediately? Wait and refresh wallet
  • Placeholder image? IPFS sync may take a moment
  • Claim link not working? Links are single-use

You focus on the event. Rekord handles the proof.
